# carousel

This is a carousel plugin.

##Usage

```html
<link rel="stylesheet" href="carousel.css">
<script src="./carousel.js"></script>
```

Then, in your code:

```html
<div class="carousel">
    <div class="carousel-wrap">
        <div class="carousel-item"><img src="./img/aa.jpg"></div>
        <div class="carousel-item"><img src="./img/bb.jpg"></div>
        <div class="carousel-item"><img src="./img/cc.jpg"></div>
        <div class="carousel-item"><img src="./img/dd.jpg"></div>
    </div>
    <div class="carousel-nav">
        <ul>
            <li><a href="#"></a></li>
            <li><a href="#"></a></li>
            <li><a href="#"></a></li>
            <li><a href="#"></a></li>
        </ul>
    </div>
</div>
```
